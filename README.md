# Keystroke-Counter
A simple C# app that reads and counts keyboard keystrokes.

## Requirements:
* Visual Studio 2017
* MetroFramework Modern UI installed

## How to add keys:
Navigate to Keystroke_Counter.cs and add the following with desired Key
```
gkh.HookedKeys.Add(Keys.A);
```
